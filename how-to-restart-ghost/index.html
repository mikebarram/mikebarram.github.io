How to restart Ghost

09 December 2016

If you're running Ghost locally and have rebooted your computer since you last used it, you'll need to restart Ghost to get it working again at http://127.0.0.1:2368/.
To do this under Windows, open the Node.js command prompt and then type
npm start

If you want it Ghost to run automatically, restarting on reboot, look here http://docs.ghost.org/pl/installation/deploy/
