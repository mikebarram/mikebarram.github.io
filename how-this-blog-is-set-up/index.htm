How this blog is set up

02 December 2016

The hosting, as you can see from the URL, is on github.io - how to publish a site under github.io

So, following that guide, I managed to create a simple home page. As github.io doesn't allow server side code for templating a site, I wanted a CMS / blogging tool that would help me manage content, wrap it in a template and produce a static site that could be synced to GitHub. By looking at what other people used, I picked ghost but not the hosted version. Instead I picked the version you download and install on Windows. This involved installing Node.js along the way.

Finally, there's the matter of syncing the content from Ghost to GitHub...
The tricky bit is that Ghost doesn't publish the content to a set of static files. Instead, you need a tool that will spider the site and save the contents to the folder that is acting as your local GitHib repository.
The standard tool in this situation, seems to be the python script buster
http://leftofnull.com/2014/02/24/using-github-pages-with-ghost-and-buster-on-windows-part-2/
